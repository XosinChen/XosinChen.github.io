  • 解决从github clone文件或者是仓库过慢的问题:

    只需要将 www.github.com/后面为代码库 改为

    www.github.com.cnpmjs.org/后面为代码库地址 就可以实现一键式加速。

打开git -bash窗口:

  • 首先进入blog所在的磁盘 cd f :/blog
  • hexo s -g 打开本地博客页面
  • hexo clean 清楚缓存,方便查看更改
  • 配置主题改blog下的.config.yml
  • 配置主题的界面更改主题下的.config.yml
  • 修改完本地上传到远端 hexo d
  • 配置ssh免密如果是https格式的网址开头记得修改成:git@github.com:XosinChen/XosinChen.github.io.git
  • 修改主题文件夹为nest就会是初始的界面
